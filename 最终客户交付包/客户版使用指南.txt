木模板开料机 G-code生成程序 - 客户版使用指南

===========================================
两种交付方案对比
===========================================

方案A：简化Python版本（推荐）
优点：
- 文件小（几MB），传输方便
- 启动快，兼容性好
- 自动安装依赖，无需手动配置
- 支持Windows/Mac/Linux多平台

使用方法：
1. 双击"简化客户版.py"
2. 程序自动检查和安装依赖库
3. 自动启动Web服务并打开浏览器
4. 界面和功能与在线版本完全一致

方案B：打包exe版本
优点：
- 客户无需安装Python
- 双击即用，更像传统软件

缺点：
- 文件大（50-100MB）
- 只能在Windows上运行
- 需要在Windows环境下打包

===========================================
推荐的客户交付流程
===========================================

第一步：发送简化版本
将以下文件打包发送给客户：
- 简化客户版.py（启动程序）

客户双击"简化客户版.py"即可开始测试：

第三步：功能验证
客户可以：
1. 上传他们的DXF文件进行转换测试
2. 调整加工参数（厚度、转速等）  
3. 下载生成的NC文件验证格式
4. 在实际机床上测试兼容性

===========================================
技术要求
===========================================

客户电脑环境要求：
- Windows 7/10/11 或 macOS 10.14+ 或 Linux
- Python 3.7或更高版本（如果没有会自动引导安装）
- 网络连接（用于自动安装依赖库）
- 2GB可用内存

如果客户电脑没有Python：
1. 访问 python.org 下载安装Python
2. 安装时勾选"Add Python to PATH"选项
3. 或者我们可以提供Python安装包

===========================================
常见问题处理
===========================================

Q1: 双击py文件没有反应
A1: 可能Python没有正确安装或关联
    解决：右键点击文件 → 打开方式 → 选择Python

Q2: 提示"ModuleNotFoundError"
A2: 依赖库安装失败
    解决：手动执行 pip install flask ezdxf

Q3: 浏览器没有自动打开  
A3: 手动打开浏览器访问 http://localhost:5001

Q4: 防病毒软件阻止运行
A4: 将程序文件夹添加到防病毒软件白名单
